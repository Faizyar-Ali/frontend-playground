# Frontend Playground

This repository is a controlled environment for learning and experimenting with modern frontend development using React + Vite.

## Purpose

- Learn React fundamentals
- Test UI concepts
- Experiment with frontend patterns
- Practice API integration (later phase)

This is NOT a production project.

## Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Node.js
